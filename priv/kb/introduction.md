## How these docs are organized

- **Skills** — languages, frameworks, and tools, with honest proficiency notes.
- **Projects** — case studies, starting with this platform itself.
- **Career** — work history and achievements, in more depth than a resume allows.
- **Personal** — hobbies, interests, and the day-to-day setup.
- **Meta** — the [colophon](/docs/colophon): how this whole site is built.

## Ask DylanBot

Every page on this site has a chat widget in the bottom-right corner. It's grounded in
these exact docs — ask it about Dylan's skills, this platform's architecture, or
anything else covered here, and it answers from what's actually written, with links
back to the source doc. If it doesn't know something, it says so rather than guessing.
